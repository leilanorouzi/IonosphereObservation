Module codes
============

There are several files including classes, functions and required parameters as follow:

*   **reading\_files**: To read antenna and source information
*   **source.py**: To obtain and calculate the parameters of the source/s
*   **antenna.py**: To obtain and calculate the parameters of the antennas
*   **source\_antenna.py**: To calculate the parameter related to the source/s and antenna/s
*   **geometry**: To calculate reference frame system  and vector rotation.
*   **visualization**: Includes functions to plot all necessary graphs, figures and diagrams.
*   **wave\_propagation** : The main class for calculations
