Module codes
============

There are several files including classes, functions and required parameters as follow:

*   **multi\_source** : The main class for calculations
*   **parameters** : Contains the media parameters. In future it will be filled by codes to calculate values for medium parameter such as refractive index, electron density ans so on.
*   **reading_files**: To read antenna and source information
*   **reference_transformation**: To calculate reference frame system  and vector rotation.
*   **visualization**: Includes functions to plot all necessary graphs, figures and diagrams.
