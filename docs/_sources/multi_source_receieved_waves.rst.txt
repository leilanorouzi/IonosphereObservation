multi\_source\_receieved\_waves module
======================================


The main code to run other modules and plot the graphics. After importing input data, you can call :py:class:`visualization` to plot or :py:class:`multi_source` to calculate different results.

Import data
+++++++++++
There are 2 ways to import the data

1- You can select the data folder. When you ru the code, a window will open and asks you to choose the inout data folder. It runs :py:fun:`multi_source_receieved_waves.select_directory` function.

2- or you can type it in here directly.
If you would like to use this method please, comment above line and uncomment following line and replace the directory path with the location of the input data in you machine.
``input_dir = "../IonosphereObservation/Data/Input/"``
